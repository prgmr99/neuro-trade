## 1. 매수/매도 UI/UX 개선

**문제:** 매수/매도 버튼이 직관적이지 않음

**개선 방향:**
- 매수 버튼 클릭 시 몇 개를 살 수 있는지 안내
- 현재 보유 캐시 기준으로 최대 매수 가능 수량 표시
- 매도 시 보유 수량 대비 매도 가능 수량 명확히 표시
- 수량 입력 UX 직관적으로 개선

## 2. 거래 피드백 (토스트 + 애니메이션 + 사운드)

**문제:** 매수/매도 완료 시 시각적/청각적 피드백이 없음

**개선 방향:**
- 매수/매도 성공 시 토스트 팝업 표시
- 토스트에 거래 내용 요약 (종목, 수량, 금액)
- 등장/퇴장 애니메이션 적용
- 거래 완료 사운드 효과 추가
