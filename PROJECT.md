# Project: NeuroTrade Multiplayer Mode

## Architecture
- `src/hooks/useRoomBattle.ts`: Manages Realtime channel connection, room state (DB sync), and presence.
- `src/components/RoomBattle/RoomBattle.tsx`: UI component for the Room Battle lobby and gameplay.
- `src/store/gameStore.ts`: Local game state manager (zustand). Currently decoupled from `useRoomBattle` day advancements, which is a bug.
- `scripts/test_multiplayer_bots.js`: The verification script (to be written) simulating concurrent clients.
